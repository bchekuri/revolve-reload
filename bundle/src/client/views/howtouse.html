Start

Click On the "Light Sea Green" extension button to start "Revolve & Reload". Extension button color will get changed to "Blue" color button.

Stop

Double Click On the "Blue" extension button to stop "Revolve & Reload". One Click on "Blue" button will take you to pause mode, after pause interval is done it will resume.

Pause

This mode will pause "Revolve & Reload" for specified interval, In this mode extension button will be in "Light Orange" color. When we are in pause mode, click on "Light Orange" extension button will stop "Revolve & Reload" completely.
